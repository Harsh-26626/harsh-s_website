TradeCraft

TradeCraft allows users to trade stocks in real time prices with a $10,000 virtual balance. Built using Flask, Jinja, HTML/CSS/JS, and SQLite3. Practice the market. Sharpen your strategy.

Project Details

TradeCraft - Invest with logic, not luck.

This platform simulates real-time stock trading using live prices, giving users a $10,000 virtual balance to manage. Users can buy/sell stocks, track their portfolio, and review transaction history‚Äîall without spending a dime.

It was built as part of CS50x from Harvard, using Flask, Jinja templates, and SQLite3 for secure data handling. Every feature‚Äîfrom stock lookup via API to hashed passwords‚Äîwas developed independently as part of the course's finance problem set.

Whether you're just curious about the stock market or want to test your strategies, this project offers a safe and engaging environment to practice.
